# VERTAMA Hugo Website

Moderne, komponentenbasierte Hugo-Website für VERTAMA GmbH mit vollständigen Produktseiten.

## 🚀 Quick Start

### Installation

1. **Hugo installieren:**
   ```bash
   # macOS
   brew install hugo
   
   # Windows (mit Chocolatey)
   choco install hugo-extended
   
   # Linux
   snap install hugo
   ```

2. **Development Server starten:**
   ```bash
   cd hugo-vertama
   hugo server -D
   ```
   
   Website läuft auf: `http://localhost:1313`

3. **Production Build:**
   ```bash
   hugo
   ```
   
   Generiert statische Dateien in `/public`
